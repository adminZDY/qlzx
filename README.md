 千里之行电商项目

 使用myeclipse（IDE）开发!
 
 
 1.环境搭建
 jdk7+,tomcat7+,sqlservice
 
 2.导入项目
 先在ide创建一个工作空间，然后找到工作空间，直接将项目复制到那个工作空间目录下
 将sql/qlzx.sql脚本在sqlservice管理工具中执行
 
 3.运行
 打开tomcat服务，访问
 http:localhost:8080/qlzx/index.jsp（前台页面）
 
 http:localhost:8080/qlzx/admin/index.jsp(后台页面)
